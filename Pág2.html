Lista com as principais tags e códigos HTML

Temos aqui um exemplo de estrutura básica de linguagem, com as principais tags e códigos HTML para sites.
Elas devem ser inseridas nesta ordem, para que a linguagem dê certo no resultado final, quando acessar o navegador.

  1. < !DOCTYPE html >
  2. < html lang-"pt-br" >
  3. < head >
  4. < meta charset="UTF-8"/ >
  5. < title> Exemplo de título< /title >
  6. < /head >
  7. < body >
  8. < p>Conteúdo < /p>
  9. < /body >
  10. < /html >
    11.

Um documento HTML é composto por tags, as quais possuem um nome e aparecem entre os sinais < >, como vimos acima, por exemplo, em < html > e < head >.
Também vimos que algumas tags precisam ser abertas e fechadas, como em < body > e < / body>. Nesse caso, a tag de fechamento deve conter a barra / antes do nome.
Outras, porém, NÃO precisam ser fechadas, como a tag < meta charset="UTF-8"/ >. Nesses casos, a adição da barra / no final da própria tag é opcional.

Além dessas características, as tags também possuem atributos, como vemos na tag < meta charset="UTF-8"/ >, que possui o atributo charset=”UTF-8”. Essas propriedades definem algumas características adicionais de cada tag e em alguns casos são obrigatórias. Seus valores devem aparecer entre aspas duplas, como no exemplo acima, ou em aspas simples, caso o próprio valor contenha aspas duplas.
