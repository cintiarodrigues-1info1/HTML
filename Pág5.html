Tabelas no HTML

Tabelas são elementos utilizados com frequência para exibir dados de forma organizada em linhas e colunas.

No HTML, elas são formadas por três tags básicas:
< table>, para delimitar a tabela;
< tr>, para indicar as linhas;
< td> para formar as colunas.

Aqui vemos um exemplo simples de tabela com 5 linhas e 3 colunas, e seu código pode ser visto logo abaixo.
